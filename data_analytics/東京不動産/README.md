# 東京都不動産価格分析プロジェクト
Tokyo Real Estate Price Analysis
</br></br>


## 📊 プロジェクト概要

国土交通省が提供する実取引データ（**40万件以上**）を活用し、東京都の不動産市場を多角的に分析。</br>
投資判断を支援するインタラクティブダッシュボードを構築しました。</br></br>

### 🎯 主な目的

- 区域別の不動産価格トレンドの把握
- 高成長エリアの特定と分析
- 投資価値の高いエリアの推奨
- リスク・リターンの可視化</br></br>

### 環境

- Python 3.9以上
- Jupyter Notebook
- Tableau Public (可視化用)</br></br>


## 📈 主要発見
### 💰 価格トレンド

> 千代田区は分析期間範囲中に価格上昇率TOP3

- 目黒区、北区、品川区
