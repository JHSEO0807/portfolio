# 東京都 不動産価格分析（Tableau × Python）
本プロジェクトでは、東京都の不動産取引データを用いて、
㎡単価を指標として価格形成の要因を分析しました。</br>
Python（Jupyter Lab）で前処理を行い、Tableauで可視化しています。
</br></br>

## 分析の背景・目的
東京都の不動産価格は、地域、駅距離、築年数など複数の要因が複雑に影響して決まると考えられます。</br>
</br>
本分析では、以下の点を中心に確認しました。

- 区部・市部による価格水準の違い</br>
- 駅からの距離が㎡単価に与える影響</br>
- 築年数と価格の関係</br>
- 年度別の価格推移</br></br>

## 使用技術

- Python
- Jupyter Notebook
- Tableau Public (可視化用)</br></br>

### 可視化・分析結果
23区と市部を分けて平均㎡単価を比較したところ、
区部では全体的に高い価格水準が見られ、</br>
特に新宿区・世田谷区などでは㎡単価が高い傾向が確認できました。

一方、市部では価格帯が全体的に低く、
武蔵野市・三鷹市など一部の地域を除き、
区部との価格差が明確に表れています。

→ 行政区分による価格水準の違いが大きいことが分かります。

### 年度別 平均価格トレンド

年度別に平均価格の推移を確認すると、
一時的な上下はあるものの、
長期的には緩やかな上昇傾向が見られました。

このことから、
東京都の不動産価格は短期的な変動はあっても、
中長期では安定的に右肩上がりしていると考えられます。

### 駅からの徒歩時間と㎡単価の関係

最寄駅までの徒歩時間と㎡単価の関係を散布図で確認したところ、
駅から近い物件ほど、</br>
㎡単価の上限が高い駅近エリアでは価格のばらつきが大きい
駅から離れるにつれて、
</br>高価格帯の物件が減少するという傾向が確認できました。

→ 駅距離は価格に強く影響する要因であることが分かります。

### 築年数と㎡単価の関係

築年数と㎡単価の関係を見ると、
築年数が増加するにつれて㎡単価は低下する傾向が見られました。

年式30年までは価格の分散が大きく、
築年数だけでなく立地条件及びその他の条件などの要因も
価格に影響していることが分かります。

### まとめ・考察
本分析から、以下の点が確認できました。

不動産価格は、単一の要因ではなく
地域・立地・建物特性・時間軸が複合的に影響しています。

平均値だけでなく、分布やばらつきを確認することで実態に近い傾向を把握できます。

→ 行政区分による価格水準の違いが大きいことが分かります。
