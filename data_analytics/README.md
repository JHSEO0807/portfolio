# Data Analysis Portfolio

本ポートフォリオは、Pythonベースで進めたデータ分析プロジェクトを整理した空間です。</br>
様々なドメインのデータを収集·前処理·視覚化した経験を整理し、分析過程で得たインサイトも一緒に記録しました。</br>
</br></br>

>🔍 Overview

- 사용 언어 : Python (pandas, numpy, matplotlib, seaborn)

- 分析環境 : JupyterLab

- 主要ドメイン : 旅行·地域・不動産データ、金融·株式/暗号通貨データ </br></br>

> 主な目標:

- データに基づいて意味のあるパターンとインサイトを導出

- ダッシュボード·BI形態の視覚化を通じて結果を直観的に伝達

- ポートフォリオとして活用できる分析プロセスの確立
