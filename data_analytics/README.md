# 東京都不動産価格分析プロジェクト
Tokyo Real Estate Price Analysis
</br></br>


## 📊 プロジェクト概要

国土交通省が提供する実取引データ（**40万件以上**）を活用し、東京都の不動産市場を多角的に分析。</br>
投資判断を支援するインタラクティブダッシュボードを構築しました。</br></br>

### 🎯 主な目的

- 区域別の不動産価格トレンドの把握
- 高成長エリアの特定と分析
- 投資価値の高いエリアの推奨
- リスク・リターンの可視化</br></br>

### 環境

- Python 3.9以上
- Jupyter Notebook
- Tableau Public (可視化用)</br></br>


## 📈 主要発見
### 💰 価格トレンド

> **千代田区**は分析期間範囲中に**15%以上の価格上昇**を記録

- 全体平均: 年率**2-3%の安定成長**
- 最高価格エリア: 千代田区、港区、渋谷区
- 最も手頃なエリア: 江戸川区、葛飾区、足立区
</br></br>
