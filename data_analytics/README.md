# Data Analysis Portfolio

本リポジトリは、Python を用いて実施したデータ分析プロジェクトをまとめたものです。</br>
旅行・地域・不動産データや金融・暗号資産データなど、複数の領域に関する分析を通じて得られた知見を整理しています。</br>
</br></br>

>🔍 Overview

- 使用言語 : Python (pandas, numpy, matplotlib, seaborn)

- 分析環境 : JupyterLab

- 主要ドメイン : 旅行·地域・不動産データ、金融·株式/暗号通貨データ </br></br>

> 主な目標

- データを元にしたパターンの把握とインサイトの抽出

- BIツール・可視化を通じた分かりやすい分析結果の提示

- ポートフォリオとして活用できる分析プロセスの確立
